TrioSage-AI is an advanced web-based multi-agent decision support system designed for conflict resolution and high-stakes scenarios. Users describe a complex problem, which is then sequentially analyzed by three distinct AI agents—each grounded in foundational literature on psychology, behavioral economics, and leadership. By forcing these agents to generate specialized insights before a final Synthesizer agent aggregates them, the system completely avoids single-agent anchor bias. The application features an elegant Streamlit interface, secure session-based API key management, and robust pipeline orchestration, delivering deeply empathetic, mathematically grounded, and actionable guidance for its users.
