The Automated Defibrillator System bridges software and embedded hardware for safe arrhythmia detection and response. ECG signals are processed in real-time by a desktop application that calculates BPM and identifies tachycardia or bradycardia events. Upon detecting an arrhythmia, the system commands an Arduino-controlled circuit to deliver a controlled low-voltage shock. The system includes temperature monitoring, GUI visualization, and logging, providing a safe, educational environment for understanding biomedical signal processing and hardware integration.
