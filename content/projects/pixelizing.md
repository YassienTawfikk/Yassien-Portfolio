Pixelizing is a Python-based image processing application built with OpenCV and PyQt. It provides tools for noise addition, filtering, edge detection, image equalization, and grayscale conversion. The toolkit emphasizes modularity through Object-Oriented Programming, allowing easy extension of new algorithms. Users can visualize effects in real time via a GUI and evaluate processing results. The project is ideal for educational purposes and practical demonstrations of fundamental image processing techniques.
