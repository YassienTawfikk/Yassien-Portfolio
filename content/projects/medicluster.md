MediCluster is a healthcare analytics project that segments patient datasets into meaningful clusters using K-Means and PCA. It preprocesses patient data with encoding, scaling, and weighted features, then identifies optimal clusters. PCA visualizations help in interpreting patient subgroups. The project also provides a joblib inference pipeline to predict clusters directly from new patient input, making it a complete unsupervised learning solution for medical data analysis.
